# Dataset

* 1 Minggu:
    * 7  x 24  x 30 = 5.040 dataset
    * 5  x 24  x 25 = 3.000 dataset
    * 7  x 8  x 30 = 1.680 dataset
    * 5  x 8  x 25 = 1.000 dataset

* 1 Tahun:
    * 52 x 5040 = 262.080 dataset
    * 52 x 3000 = 156.000 dataset
    * 52 x 1680 = 87.360 dataset
